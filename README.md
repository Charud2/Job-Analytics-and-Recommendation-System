# Job-Analytics-and-Recommendation-System
A python project that analyzes job data to identify in-demand skills, salary trends, and location-based insights.

# Features
- Analyse job trends by location
- Identify top in-demand skills
- Top skills analysis
- Location-based job trends
- Average salary insights

# Tech Stack
- Python
- Pandas
- Matplotlib

# How to Run
pip install pandas matplotlib
python main.py

# Sample Insights
- Python is the most in-demand skill in Gurugram
- Average salary: Rs 5.5LPA
